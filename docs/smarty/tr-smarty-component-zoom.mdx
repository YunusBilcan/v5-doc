Ürün detay sayfasında ürün resmi için kullanılır.
Sadece masaüstünde çalışır.
<code>data-target="#product-right"</code> yazılınca zoom için target elementi belirtir. default değeri "#product-right"dır.
<p>Herhangi bir yerde kullanılabilir.Ürün detay sayfası olması gerekmez. </p> 



```html title="Örnek kullanım" 
<figure class="image-zoom h-100">
    <img src="image-path" alt="img-alt" data-toggle="zoom-image" data-target="#product-right" data-width="450" data-height="450" data-image="zoom-image-path">
</figure>
```
