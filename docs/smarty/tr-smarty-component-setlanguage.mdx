---
title: SetLanguage
---
Site dilini değiştirmek için kullanılır

```html title="Açılır menü şeklinde kullanımı" 
<select class="form-control" data-toggle="language">
    <option value="tr">Türkçe</option>
    <option value="en">English</option>
</select>
```

```html title="Buton şeklinde kullanımı" 
<a href="javascript:void(0);" data-toggle="language" data-language="tr">Türkçe</a>
<a href="javascript:void(0);" data-toggle="language" data-language="en">English</a>
```
