---
title: Placeholder
---
Yer tutucu bileşeni.

Form input elementlerinde kullanılan placeholder özelliği için geliştirilmiş, yumuşak geçiş özellikli yer tutucu.

Placeholder tanımlı input, bir üst kapsayıcısı baz alarak geçiş aksiyonunu sağlamaktadır. 
Bu yüzden bir div veya farklı bir element içerisinde kullanımı, sağlıklı çalışmasını sağlayacaktır.

```html title="Örnek kullanım"
<div>
    <input type="text" placeholder="Adınız" data-toggle="placeholder">
</div>
```
