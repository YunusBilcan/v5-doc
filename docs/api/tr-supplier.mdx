---
title: Tedarikçi
---

Tedarikçi servisi sabit alanı


```json
/srv/service/supplier/
```

### Tedarkçi mesaj kayıt
```json
/srv/service/supplier/register/

// post:  title password username, address, email, required
// all:  title, username, password, email, mobile_phone, home_phone, fax, tax_office, tax_number, authorized_person, iban, address, city, town, field1

```
### Tedarkçi bilgi 
```json
/srv/service/supplier/get-info/{ID=0}
/srv/service/supplier/get-info/1


```
### Tedarkçi mesaj gönderme 
```json
/srv/service/supplier/send-message/
// post: subject, message, security_code, supplierId


```
