title: Kampanya Hopi
---
Kampanya hopi servisi sabit alanı
```json
/srv/service/hopi/

```



### Hopi kampanya üye giriş kontrolü
```json
/srv/service/hopi/signin/

```
### Hopi kampanyaları görme
```json
/srv/service/hopi/get-campaigns/

```
### Hopi kampanyalar oluşturma
```json
/srv/service/hopi/set-campaign/
// post campaign coin

```
### Hopi kampanya iptal etme
```json
/srv/service/hopi/remove-campaign/

```
