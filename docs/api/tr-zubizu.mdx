---
title: Zubizu kampanya 
---

Zubizu kampanya servisi sabit alanı


```json
/srv/service/zubizu/
```

### Giriş 
```json
/srv/service/zubizu/signin/
```

### Çıkış 
```json
/srv/service/zubizu/signout/

```
### Özet
```json
/srv/service/zubizu/summary/

```

### Kampanyalar
```json
/srv/service/zubizu/get-campaigns/

```
### Kampanya oluşturma
```json
/srv/service/zubizu/set-campaign/{GROUP}/{CAMPAIGN_ID}/
// /srv/service/zubizu/set-campaign/5/7/

```
### Kampanya iptali
```json
/srv/service/zubizu/unset-campaign/{CAMPAING_ID}/
/srv/service/zubizu/unset-campaign/7/

```
