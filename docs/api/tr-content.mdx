title: İçerik Servisi

İçerik sayfası servisi sabit alanı
```json
/srv/service/content/
```



### Blok sayfası servisi
```json
/srv/service/content/get/{BLOCK_ID}/{FOLDER}/{PARAMS}
// /srv/service/content/get/1004/beni-ara
// /srv/service/content/get/1110/set-location


```
### İçerik  blok servisi
```json
/srv/service/content/get-block/{BLOCK_ID}/{TYPE}/{PARAMS_ID}
/srv/service/content/get-block/1018/content/1



```
### Sözleşmeler  blok servisi
```json
/srv/service/content/get-setting/{TITLE}
// /srv/service/content/get-setting/KvkkMesaj
// 'KvkkMesaj', 'UyelikSozlesmesi', 'SmsTicariOnayMetni', 'EmailTicariOnayMetni', 'TelefonTicariOnayMetni'  //izin verilen alanlar




```
