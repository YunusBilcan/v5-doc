---
Kişiselleştirme servisi sabit alanı
```json
/srv/service/personalization/

```



### Kişiselleştirme çağırma
```json
/srv/service/personalization/{PERSONALIZATION_ID}/{CART_ID}/{PRODUCT_ID}
/srv/service/personalization/get/4/-1/146


```

### Kişiselleştirme sıralama
```json
/srv/service/personalization/sort-by-key/{DATA}/{SORT_KEY}/{SORT_FLAG=SORT_ASC}/
//kullanılmamış


```

### Kişiselleştirme kaydetme
```json
/srv/service/personalization/save/{CART_ID/INDEX}/
//kullanılmamış


```

### Kişiselleştirme sepete ekleme (Lisansa bağlı)
```json
/srv/service/personalization/add-to-cart/{TOKEN}/{REDIRECT=0}/



```
