title: Adres Servisi
---

Ürünler servisi sabit alanı
```json
/srv/service/address/
```

### Üye dres bilgisi servisi
```json
/srv/service/address/get-list/{$PAGE} 
//üye girişi gerekir. order, home gibi

```
### Adres bilgisi servisi
```json
/srv/service/address/get/0/1/order
//0=id,1=uppercase,page
```


### Adres bilgisi silme
```json
/srv/service/address/delete/1/
//1=adress id
```


### Adres bilgisi kaydetme
```json

/srv/service/address/save/1/
//1=id

```
