---
title: Ürünler Servisi
---

Ürünler servisi sabit alanı

```json
/srv/service/product/
```

### İlgili (ilişkili) ürün servisi
```json
/srv/service/product/get-related-products/{$P.ID}/1
```
### İkame ürüm servisi
```json
/srv/service/product/get-related-products/{$P.ID}/2
```
### Aksesuar ürün servisi
```json
/srv/service/product/get-related-products/{$P.ID}/3
```
### Tamamlayıcı ürün servisi
```json
/srv/service/product/get-related-products/{$P.ID}/4
```
### Alt ürün resimleri servisi
```json
/srv/service/product/get-sub-product-images/{$P.ID}/{$P.VARIANT_ID}
```

### Ürün bilgiler servisi
Kullanılabilir Parametreler:
'DOCUMENT_INFO', 'DETAIL', 'BARCODE', 'WARRANTY_INFO', 'DELIVERY_INFO', 'NOTES', 'PERSONALIZATION_ID'

```json
/srv/service/product/get/{$P.ID}/{PARAMS}
/srv/service/product/get/{$P.ID}/document_info-detail-barcode-warranty_info-delivery_info-notes-personalization_id

```
`

### İlk alt ürüne göre ikinci alt ürün servisi
```json
/srv/service/product/get-sub-two/{$P.ID}/{$P.VARIANT_ID}

```
### Alt ürün fotoğraf servisi
```json
/srv/service/product/get-sub-product-images/{$P.ID}/{$P.VARIANT_ID}

```


### Ürün arama servisi
```json
/srv/service/product/search/{WORD}

```

### Ürün JSON arama servisi
```json
/srv/service/product/search-json/{WORD}

```
### Ürün tüm arama servisi
```json
/srv/service/product/search-all/{WORD}

```
