---
title: Ürünler Servisi
---

Ürünlerle ilgili servisler ile başlamaktadır.

```json
/srv/service/product/
```

### İlgili (ilişkili) ürün servisi
```json
/srv/service/product/get-related-products/{$P.ID}/1
```
### İkame ürüm servisi
```json
/srv/service/product/get-related-products/{$P.ID}/2
```
### Aksesuar ürün servisi
```json
/srv/service/product/get-related-products/{$P.ID}/3
```
### Tamamlayıcı ürün servisi
```json
/srv/service/product/get-related-products/{$P.ID}/4
```
### Alt ürün resimleri servisi
```json
/srv/service/product/get-sub-product-images/{$P.ID}/{$P.VARIANT_ID}
```
