title: Sepet Servisi
---
Sepet servisi sabit alanı
```json
/srv/service/cart/
```





### Sepet Sayfası Bilgisi
```json
/srv/service/cart/load/
// /srv/service/cart/load?link=sepet

```
### Sepet Açılır Pencere
```json
/srv/service/cart/load-win/{P.ID}?link={LINK}
//  /srv/service/cart/load-win/153?link=test

```

### Sepet Toplu güncelleme
```json
srv/service/cart/update/{ITEM_COUNT}
// srv/service/cart/update/1-1-2
```
### Sepet ürün güncelleme
```json
srv/service/cart/update-item/{SESSION_ID}/{ITEM_COUNT}/
// srv/service/cart/update/1/5
```
### Sepet paket güncelleme
```json
/srv/service/cart/update-field/{SESSION_ID}/{FIELD}/{VALUE}
/srv/service/cart/update-field/1/HedPaket/1
SESSION:KART SIRASI
FIELD:HedPaket,SubscribeFrequency,SepetDizi 
```
### Sepet ürün silme
```json
srv/service/cart/remove/{SESSION_ID}
//srv/service/cart/remove/0
```
### Sepet ürün silme
```json
srv/service/cart/delete/{SESSION_ID}
//srv/service/cart/delete/1
```
### Sepet tüm ürünleri  silme
```json
srv/service/cart/deleteall
```
### Sepet kampanya kodu işleme
```json
/srv/service/cart/set-campaign-code/{CAMPAIGN_CODE}
// /srv/service/cart/set-campaign-code/tekrom
```
### Sepet ürün ekleme
```json
/srv/service/cart/add-to-cart/{P.ID}/{QUANTITY}/{VARIANT_ID}
// /srv/service/cart/add-to-cart 
//post
```
### Mikro site  export özelligi
```json
/srv/service/cart/export-main-site?prId={P.ID}&quantity={QUANTITY}&subId={VARIANT_ID}
//  /srv/service/cart/export-main-site?prId=151
```
### Main site import özelligi
```json
/srv/service/cart/import-micro-site?productId={P.ID}&quantity={QUANTITY}&subId={VARIANT_ID}
//  /srv/service/cart/import-micro-site/?productId=154&quantity=5&subId=2229
```
### Hediye notu görme
```json
/srv/service/cart/get-gift-notes/{SESSION_ID}
//  /srv/service/cart/get-gift-notes/0
```
### Hediye notu kaydetme
```json
/srv/service/cart/set-gift-notes/{SESSION_ID}
// /srv/service/cart/set-gift-notes/0
//post <content>
```
### Sipariş notu görme
```json
/srv/service/cart/get-order-notes/{SESSION_ID}
// /srv/service/cart/get-order-notes/0

```
### Sipariş notu kaydetme
```json
/srv/service/cart/set-order-notes/{SESSION_ID}
// /srv/service/cart/set-order-notes/0
//post <content>
```
### Sepet geneli sipariş notu
```json
/srv/service/cart/get-general-order-note/

```
### Sepet geneli sipariş notu kaydetme
```json
/srv/service/cart/set-general-order-note/
//post <content>
```
### Haftasonu teslimat olsun/olmasın
```json
/srv/service/cart/set-approval-box/{NUMBER}
// /srv/service/cart/set-approval-box/1  // 1 veya 0

```

### Sepet teslimat günü ekleme
```json
/srv/service/cart/set-delivery-day/{DAY}
//  /srv/service/cart/set-delivery-day/Mon  //Sun,Wed
```
### Sepet teslimat saati ekleme
```json
/srv/service/cart/set-delivery-date/{DATE}
//  /srv/service/cart/set-delivery-date/11.12.2022
```
### Sepet teslimat saati ekleme
```json
/srv/service/cart/set-delivery-hour/{HOUR}
//  /srv/service/cart/set-delivery-hour/12:55
```
### Üst açılır sepet gösterme
```json
/srv/service/cart/show-header-cart

```
### Resim notları
```json
/srv/service/cart/get-image-notes

```
### Sepet senkronize et
```json
/srv/service/cart/synchronize

```
