---
title: Kategori Servisi
---
Kategori servisi sabit alanı
```json
/srv/service/category/
```





### Kategori Bilgisi
```json

/srv/service/category/hierarchy/{CATEGORY_ID}
//  /srv/service/category/hierarchy/77


```

### Kategori Liste Bilgisi (idli)
```json

/srv/service/category/hierarchy-list/{CATEGORY_ID}
// /srv/service/category/hierarchy-list/77


```
### Alt kategori bilgisi
```json

/srv/service/category/get/{CATEGORY_ID}/{ORDERBY}
/srv/service/category/get/45/katadi


```
### Kategori ağacı bilgisi
```json

/srv/service/category/get-tree


```
