Bölge servisi sabit alanı

```json
/srv/service/region/
```

### Ülke Servisi
```json
/srv/service/region/get-countries/{NONELIMIT=0}
// /srv/service/region/get-countries/1
```

### Şehir servisi
```json
/srv/service/region/get-cities/{PARENT=TR}
// /srv/service/region/get-cities/de
```
### İlçe servisi
```json
/srv/service/region/get-towns/{PARENT=46}
// /srv/service/region/get-towns/46
```
### Semp servisi
```json
/srv/service/region/get-districts/{PARENT=0}
// /srv/service/region/get-districts/0
```
### Tüm servisler tek tek
```json
/srv/service/region/get-list/{TYPE}/{PARENT}/{WITH_LIMIT}
// /srv/service/region/get-list/S/TR/0
// /srv/service/region/get-list/I/35/0
// /srv/service/region/get-list/M/988/0
```
