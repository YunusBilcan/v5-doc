Dil servisi sabit alanı
```json
/srv/service/language/

```



### Dil Dosyaları
```json
/srv/service/language/get-content/{PARAMS}
// /srv/service/language/get-content/address/
//  CCpanel CurrentPriceBox Address AddressAdd Init Success State Campaigns 

```
### Dil ve parabirimi değiştirme yönlendirme
```json
/srv/service/language/redirect/{LANGUAGE}/{CURRENCY}
// /srv/service/language/redirect/tr/TL

```
### Dil değiştirme
```json
/srv/service/language/change/{LANGUAGE}/{TYPE}/{TABLE_ID}/
// /srv/service/language/change/tr/product/146

```
### Seçilebilir dil listesi
```json
/srv/service/language/get-list/

```
### Seçili dil 
```json
/srv/service/language/get-language/

```
