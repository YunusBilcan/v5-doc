İnteraktif vitrin servisi sabit alanı

```json
/srv/service/showcase/
```

### Tablı vitrin
```json
/srv/service/showcase/tab/{BLOCK_ID}/{TAB_ID}/
// /srv/service/showcase/tab/1055/1
```
### Interaktiv vitrin
```json
/srv/service/showcase/interactive
```
### Interaktiv mobil vitrin
```json
/srv/service/showcase/interactive-mobile
```
