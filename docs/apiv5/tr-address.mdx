## Adresleri listele

```json
{{API_URL}}/api/v1/public/address?debug=

// get
```

## IP ile adres getir

```json
{{API_URL}}/api/v1/public/address/1

// get
```
## Adres gereksinimlerini listeleme

```json
{{API_URL}}/api/v1/block/get-page-requirements

// post > page : customer-insert-address-modal
```

## Adres ekleme

```json
{{API_URL}}/api/v1/block/get-page-requirements

// post > 
// country_code, address, city, district, fullname, identity_number, mobile_phone, nationality, title, nationality, town,
// town_code,  district_code, province, email

```
## Adresi kurtar

```json
{{API_URL}}/api/v1/public/address/1/recover

// post > address

```
## Teslimat adres seçimi

```json
{{API_URL}}/api/v1/public/address/delivery/:id

// post > 
// country_code, address, city, district, fullname, identity_number, mobile_phone, nationality, province, town, title


```
