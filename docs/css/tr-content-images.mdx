---
title: Görüntüler / Resimler
---

Tarayıcı ölçeğine duyarlı görüntüler, resimler vb. medyaları kullanırken sizler için 
önceden <code>img</code> seçicisine <code>max-width</code> değerini tanımladık.

### Çözünürlüğe duyarlı görsel

Medyalara herhangi bir class yazmadan doğrudan duyarlı hale getirilebilir. 
Maksimum genişlik için: %100; ve yükseklik: otomatik; olacak şekilde ayarlayacaktır.

Buna ek olarak kapsayıcı genişliğini almasını isterseniz <code>w-100</code> verebilirsiniz.

<div class="doc-image-example mt-2">
  <svg class="w-100" width="100%" height="250" xmlns="http://www.w3.org/2000/svg" role="img" aria-label="Placeholder: Responsive image" preserveAspectRatio="xMidYMid slice" focusable="false"><title>Placeholder</title><rect width="100%" height="100%" fill="#2a2d3e"></rect>
    <text x="50%" y="50%" fill="#fd6905" dy=".3em">v5 Responsive image</text>
  </svg>
</div>

```html title="html"
<img src="..." alt="...">
```
