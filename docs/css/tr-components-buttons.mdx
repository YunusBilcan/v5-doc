## Durumlar

### Devre dışı 

Herhangi bir <code>&lt;button&gt;</code> öğesine devre dışı bırakılmış niteliğini ekleyerek 
buton devre dışı görünmesini sağlayın. Devre dışı bırakılan düğmelerin işaretçi olayları vardır: 
hiçbiri uygulanmaz, vurgulu ve etkin durumların tetiklenmesi engellenir. 

<div class="px-0 mb-1 border bg-light bdr-2">
  <div class="row m-0 p-2 w-100">
    <button type="button" class="btn btn-primary" disabled>Primary button</button>
    <button type="button" class="btn btn-secondary ml-1" disabled>Button</button>
  </div>
</div>

```html
<button type="button" class="btn btn-primary" disabled>Primary button</button>
<button type="button" class="btn btn-secondary" disabled>Button</button>
```

##### &lt;a&gt; öğesini kullanan devre dışı bırakılan butonlar farklı davranır:
* <code>&lt;a&gt;</code> disable niteliğini desteklemediğinden, görsel olarak devre dışı görünmesi için <code>.disabled</code> sınıfını eklemelisiniz.
* Bağlantı butonlarına tüm <code>pointer-events</code> devre dışı bırakmak için bazı stiller dahil edilmiştir.

<div class="v5-example-alert-info text-black">
  Duyarlı bir tasarım için <code>aria-disabled="true"</code> içermesi tavsiye edilir.
</div>

<div class="px-0 mb-1 border bg-light bdr-2 mt-3">
  <div class="row m-0 p-2 w-100">
    <a href="#" class="btn btn-primary disabled" aria-disabled="true">Primary link</a>
    <a href="#" class="btn btn-secondary disabled ml-1" aria-disabled="true">Link</a>
  </div>
</div>

```html
<a href="#" class="btn btn-primary disabled" aria-disabled="true">Primary link</a>
<a href="#" class="btn btn-secondary disabled" aria-disabled="true">Link</a>
```

### Aktif

Herhangi bir <code>&lt;button&gt;</code> öğesine veya <code>&lt;a&gt;</code> etiketine 
aktif niteliğini ekleyerek butonun aktif duruma geçmesini sağlayın. 

<div class="px-0 mb-1 border bg-light bdr-2">
  <div class="row m-0 p-2 w-100 justify-content-evenly">
    <div class="my-md-1">
      <strong>Buton:</strong>
      <button type="button" class="btn btn-primary active ml-1">Primary button</button>
      <button type="button" class="btn btn-secondary active ml-1">Button</button>
    </div>
    <div class="my-md-1">
      <strong>a tag:</strong>
      <a href="#" class="btn btn-primary active ml-1">Primary link</a>
      <a href="#" class="btn btn-secondary active ml-1">Link</a>
    </div>
  </div>
</div>

```html
<button type="button" class="btn btn-primary active">Primary button</button>
<button type="button" class="btn btn-secondary active">Button</button>

<a href="#" class="btn btn-primary active">Primary link</a>
<a href="#" class="btn btn-secondary active">Link</a>
```
