# - Hizalama
Sütunları dikey ve yatay olarak hizalamak için esnek kutu hizalama yardımcı classlarını kullanın.

## Dikey Hizalama

### .align-items-flex-start
(varsayılan) Öğeleri başlangıç noktasında (sola dayalı) hizalama.

<div class="px-0 mb-1 border bg-light bdr-2 v5-example-layout-alignment">
  <div class="row m-0 p-2 w-100">
    <div class="col border bg-dark text-white p-3 bdr-2 text-center">
      Dikey Hizalama - Başlangıç
    </div>
  </div>
</div>

```html
<div class="d-flex align-items-flex-start">
  ...
</div>
```

### .align-items-center
Öğeleri ortada hizalama.

<div class="px-0 mb-1 border bg-light bdr-2 v5-example-layout-alignment d-flex align-items-center">
  <div class="row m-0 p-2 w-100">
    <div class="col border bg-dark text-white p-3 bdr-2 text-center">
      Dikey Hizalama - Orta
    </div>
  </div>
</div>

```html
<div class="d-flex align-items-center">
  ...
</div>
```

### .align-items-flex-end
Öğeleri bitiş noktasında (sağa dayalı) hizalama.

<div class="px-0 mb-1 border bg-light bdr-2 v5-example-layout-alignment d-flex align-items-flex-end">
  <div class="row m-0 p-2 w-100">
    <div class="col border bg-dark text-white p-3 bdr-2 text-center">
      Dikey Hizalama - Son
    </div>
  </div>
</div>

```html
<div class="d-flex align-items-flex-end">
  ...
</div>
```

### .justify-content-evenly (eksik eklenecek)
Tüm öğelerin arası aynı eşitlikte olacak şekilde hizalama.

<div class="px-0 mb-1 border bg-light bdr-2">
  <div class="row m-0 p-2 w-100 justify-content-evenly">
    <div class="col-auto border bg-dark text-white p-3 bdr-2 text-center">
      Yatay Hizalama
    </div>
    <div class="col-auto border bg-dark text-white p-3 bdr-2 text-center">
      Yatay Hizalama
    </div>
  </div>
</div>

```html
<div class="d-flex justify-content-evenly">
  ...
</div>
```

### .justify-content-around
İlk ve son öğe hariç diğerlerinin arası eşit olacak şekilde hizalama.

<div class="px-0 mb-1 border bg-light bdr-2">
  <div class="row m-0 p-2 w-100 justify-content-around">
    <div class="col-auto border bg-dark text-white p-3 bdr-2 text-center">
      Yatay Hizalama
    </div>
    <div class="col-auto border bg-dark text-white p-3 bdr-2 text-center">
      Yatay Hizalama
    </div>
  </div>
</div>

```html
<div class="d-flex justify-content-around">
  ...
</div>
```

### .justify-content-between
İlk öğe sola dayalı, son öğe sağa dayalı ve diğerleri ortalı olacak şekilde hizalama.

<div class="px-0 mb-1 border bg-light bdr-2">
  <div class="row m-0 p-2 w-100 justify-content-between">
    <div class="col-auto border bg-dark text-white p-3 bdr-2 text-center">
      Yatay Hizalama
    </div>
    <div class="col-auto border bg-dark text-white p-3 bdr-2 text-center">
      Yatay Hizalama
    </div>
  </div>
</div>

```html
<div class="d-flex justify-content-between">
  ...
</div>
```
