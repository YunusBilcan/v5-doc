### Sass
Aşağıdaki Sass işlevine ek olarak, 
renkler ve daha fazlası için dahil edilen CSS özel özelliklerimizi 
(diğer adıyla CSS değişkenleri) inceleyebilirsiniz.
### Değişkenler

```scss
$borderPrimary:     #fd6905;
$borderSecondary:   #dddddd;
$borderWhite:       #ffffff;
$borderBlack:       #000000;
$borderGray:        #a0a0a0;
$borderLight:       #f9f9f9;
$borderDarkGray:    #2c2a28;
$borderSuccess:     #44c61f;
$borderInfo:        #93d8dd;
$borderWarning:     #fcdf27;
$borderDanger:      #ff0000;
```


### Dış çizgi
Bir öğeye kenarlık ekleme, bir kenar hariç uygulama ve köşeleri yumuşatmak için kullanabileceğiniz
önceden tanımlanmış class'lar mevcuttur.

<div class="p-2 border mb-1 v5-example-border-utils">
    <div class="px-1">
      <div class="row">
        <div class="col-auto p-2">
          <div class="p-2 border border-dark bg-gray v5-example-border"></div>
        </div>
        <div class="col-auto p-2">
          <div class="p-2 border-top border-dark bg-gray v5-example-border"></div>
        </div>
        <div class="col-auto p-2">
          <div class="p-2 border-right border-dark bg-gray v5-example-border"></div>
        </div>
        <div class="col-auto p-2">
          <div class="p-2 border-bottom border-dark bg-gray v5-example-border"></div>
        </div>
        <div class="col-auto p-2">
          <div class="p-2 border-left border-dark bg-gray v5-example-border"></div>
        </div>
        
      </div>
    </div>
</div>

```html
<span class="border"></span>
<span class="border-top"></span>
<span class="border-end"></span>
<span class="border-bottom"></span>
<span class="border-start"></span>
```
