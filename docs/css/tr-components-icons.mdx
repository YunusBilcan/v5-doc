## Standart tema ikonları (opsiyon)
v5 ikonları haricinde T-Soft bünyesinde geliştirilmiş, <u>standart temalar için</u> fazladan eklenmiş veya değiştirilmiş ikonlardır. 
** Özel temaları kapsamamaktadır.** 
Projeye sonradan dahil edilerek isteğe bağlı kullanılır.

<div class="bg-yellow p-1 border border-light border-round">
  Sadece o temanın kütüphanesinde olan ikonların arkaplanı farklı renk ile vurgulanmıştır.
</div>

##### ** Kullanım durumunda: **
- Kullanmak istediğiniz temada <code>@import</code> etmeniz durumunda projenize seçtiğiniz ikon kütüphanesinin değerleri gelmektedir. 
- ** Projenize ekstra dosya dahil edilmez. **
- <u>Standart tema dizininden</u> (theme/v5/) <code>icon-font</code> ve <code>css</code> dosyaları çağırılmaktadır.
- - <code>icon-font</code> : /theme/v5/css/fonts/[tema adı]/[tema adı]-icon.woff, .woff2
- - <code>css</code> : /theme/v5/scss/icon/[tema adı]-icon.scss

### Basic teması ikonları

#### Projeye Dahil etme
```scss title="tema dizinindeki style.scss dosyasına @include BLOCK içine eklenmelidir"
@include BLOCK {
  @import "icon/basic-icon.scss";
  ...
}
```
