### Sass
Aşağıdaki Sass işlevine ek olarak, 
renkler ve daha fazlası için dahil edilen CSS özel özelliklerimizi 
(diğer adıyla CSS değişkenleri) inceleyebilirsiniz.
### Değişkenler

```scss
$bgPrimary:     #fd6905;
$bgSecondary:   #dddddd;
$bgWhite:       #ffffff;
$bgBlack:       #000000;
$bgGray:        #a0a0a0;
$bgLight:       #f9f9f9;
$bgDarkGray:    #2c2a28;
$bgSuccess:     #44c61f;
$bgInfo:        #93d8dd;
$bgWarning:     #fcdf27;
$bgDanger:      #ff0000;
```
