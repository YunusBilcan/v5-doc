### Aşağı Kaydırme Animasyonu

```html live
<div class="p-2 mb-2 bg-primary text-white rounded">
  <div class="slide-in-down text-center fw-bold">Aşağı Kaydırme</div>
</div>
```

```html
  <div class="slide-in-down">...</div>
```

### 360° Dönme Animasyonu

```html live
<div class="p-2 mb-2 bg-primary text-white rounded position-relative">
  <div class="transform-rotate text-center fw-bold">360° Dönme</div>
</div>
```

```html
  <div class="transform-rotate">...</div>
```
