### Kırılma noktası ile kullanım

Kırılma noktaları ile beraber kullanabilirsiniz. Örneğin, .w-25.w-md-100... 

```css
	
@media (min-width: 1170px) {
  .w-xl-100 {
    width: 100%;
  }
  .w-xl-75 {
    width: 75%;
  }
  ...
}
	
@media (min-width: 768px) {
  .h-md-100 {
    height: 100%;
  }
  .h-md-75 {
    height: 75%;
  }
  ...
}

...

```
