title: Notify
---
Bildiri penceresi bileşeni.

```javascript
T.notify({
    text: "T-SOFT E-TİCARET SİSTEMLERİ",
    stopOnFocus: false,
    duration: 6000,
    close: false,
    gravity: "top", // top || bottom
    position: 'right', // left || right
    className: '',
    iconClass: `ti-thumbs-up`
});
```
