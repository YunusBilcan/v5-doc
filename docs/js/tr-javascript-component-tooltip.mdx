---
title: Tooltip
---
Bilgilendirme çubuğu bileşeni.

Elementlerin title niteliğini biçimlendirerek gösterimi sağlanır. <code>data-toggle="tooltip"</code>
ön adı ile js yazmadan aktivasyonu gerçekleştirilebilir. Mouse ilgili elementin üzerine gelince görünür olur. 
<code>data-placement</code> niteliği ile elementin altında ya da üstünde konumu değiştirilebilir.

<div class="my-2">
    <code>data-placement</code> aldığı değerler: <strong>top / bottom</strong> (varsayılan:top)
</div>

```html title="Örnek kullanım" 
<a href="javascript:void(0);" data-toggle="tooltip" title="Sepete Ekle">Test tooltip</a>
```

```html title="Konum belirtme örnek kullanımı" 
<a href="javascript:void(0);" data-toggle="tooltip" data-placement="top" title="Sepete Ekle">Test tooltip</a>
```

```javascript title="Javascript ile kullanımı" 
T.tooltip(selector, options);
/*
Options is obje tipindedir ve sadece placement değeri alır.

T.tooltip(selector, {
    placement: 'top' // default değer
});
*/
```
