---
title: SetCurrency
---
Site para birimini değiştirmek için kullanılır.

```html title="Açılır menü şeklinde kullanımı" 
<select class="form-control" data-toggle="currency">
    <option value="tl">TL</option>
    <option value="usd">USD</option>
    <option value="eur">EURO</option>
</select>
```

```html title="Buton şeklinde kullanımı" 
<a href="javascript:void(0);" data-toggle="currency" data-language="tl">TL</a>
<a href="javascript:void(0);" data-toggle="currency" data-language="usd">USD</a>
<a href="javascript:void(0);" data-toggle="currency" data-language="eur">EURO</a>
```
