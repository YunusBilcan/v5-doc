Bilgilendirme çubuğu bileşeni.

Sağdan ya da soldan kayarak açılan pencere işlevi görür. a elementi pencere açımını tetiklemek için kullanılır, 
<code>href</code> niteliği baz alınır. Href niteliğine pencere elementinin id değeri verilir.

Kullanımı;

```html
<a href="#test" data-toggle="drawer">Open drawer</a>

<div data-rel="test" class="drawer-overlay"></div>
<div id="test" class="drawer-wrapper" data-display="push" data-position="left">
    <div class="drawer-close">
        <i class="ti-close"></i>
    </div>
    <!-- your content -->
</div>
```

data-display: <strong>push</strong> ya da <strong>overlay</strong> değerlerini alır. 
Push sayfayı iterek pencere açılımını, overlay ile sayfa üzerinde açılımını sağlar.

<div class="mb-2">
<code>data-position</code>: <strong>left</strong> ya da <strong>right</strong> değerlerini alır. 
Left soldan, right ise sağdan pencere açılımını sağlar.
</div>

```html
<div data-rel="test" class="drawer-overlay"></div> 
```
Bu element tanımı pencere açıldığında sayfanın css ile kararmasını sağlayabilmek için kullanılır. 
<code>data-rel</code> niteliğine pencere elementini id değeri verilir.
