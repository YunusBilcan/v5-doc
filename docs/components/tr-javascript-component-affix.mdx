---
title: Affix
---
Yapışkan bloklar oluşturmak için kullanılır.

```html title="Örnek kullanımı" 
<div data-toggle="affix">yapışkan alan</div>
```

<div class="mb-1">
Yapışkan elementin sabit kalacağı referans elementi belirtmek için 
<code>data-bottom-ref="selector"</code> kullanılabilir. 
Belirtilmediği takdirde <code>footer</code> elementi baz alınacaktır. 
</div>

Örnek kullanım;

```html
<div data-toggle="affix" data-bottom-ref="#my_element_id">yapışkan alan</div>
```
