Input file tipi için geliştirilmiş, özelleştirilebilir dosya seçici.

```html title="Örnek kullanımı"

<label for="input-file-img" class="form-control">
    <span>Dosya Seçiniz</span>
    <i class="ti ti-image-plus text-primary"></i>
</label>
```
