Countdown Bileşeni (Geri Sayım Sayacı)
Countdown bileşeni, geri sayım bileşenidir.
<div class="mb-2">
    <code>data-toggle="count-down"</code> ön adı kullanarak js yazmadan bu bileşeni 
    sayfanızda kullanabilirsiniz.
</div>

day, hour, minute ve second classı içerisindeki timer-text içerisine sonuc yazdırılmaktadır. 

```html title="Örnek kullanımı"
<div data-timer="2022,05,25,15,55,16" data-toggle="count-down">
    <div class="day">
        <span class="timer-text">00</span>
    </div>
    <div class="hour">
        <span class="timer-text">00</span>
    </div>
    <div class="minute">
        <span class="timer-text">00</span>
    </div>
    <div class="second">
        <span class="timer-text">00</span>
    </div>
</div>
```
