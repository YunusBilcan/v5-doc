title: İçerik Servisi v5
---

İçerik sayfası servisi sabit alanı
```json
/srv/service/content-v5/
```



### Alt klasör servisi v5
```json
/srv/service/content-v5/sub-folder/{PAGE_ID}/{$BLOCK_ID}/[SUB_FOLDER]?{PARAMS}
// /srv/service/content-v5/sub-folder/3/1004/installment-list?product=726


```
### İçerik yönetim servisi v5

```json
/srv/service/content-v5/fixed-page-content/{CONTENT_NUMBER}
// /srv/service/content-v5/fixed-page-content/5


```
