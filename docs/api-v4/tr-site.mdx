Site servisi sabit alanı

```json
/srv/service/site/
```

### Sipariş kargo 
```json
/srv/service/site/get-order-cargo/{ORDER_NUMBER}/
// /srv/service/site/get-order-cargo/5/

```
### Tavsiye
```json
/srv/service/site/advice/
// post: email, AliciAdi,AliciEposta, GonderenAdi, GonderenEposta, Mesaj  

```
### Çerez Politikası verileri
```json
/srv/service/site/get-cookie-law/
```


### Yorumlar
```json
/srv/service/site/get-comments/{PAGE=1}/{LIMIT=10}/
/srv/service/site/get-comments/1/1/

```
