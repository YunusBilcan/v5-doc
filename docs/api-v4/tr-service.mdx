Bölge servisi sabit alanı

```json
/srv/service/service/
```

### Sepet bilgileri
```json
/srv/service/service/get-member-current-basket/
```

### Son siparişler
```json
/srv/service/service/get-member-last-order/
```
### Kategori ağacı
```json
/srv/service/service/get-category-hierarchy/{CAT_ID}/
/srv/service/service/get-category-hierarchy/35/
```

### Üye bilgileri
```json
/srv/service/service/get-member-info/
```
### Sepet ürün sayısı
```json
/srv/service/service/get-basket-count/
```
