---
title: Ziyaretçi  
---
Ziyaretçi servisi sabit alanı
```json
/srv/service/guest/

```



### Ziyaretçi defteri yeni mesaj formu
```json
/srv/service/guest/form-new-message/

```
### Ziyaretçi defteri yeni mesaj 
```json
/srv/service/guest/send-message/
//post   guest-fullname,guest-email,guest-phone, guest-message

```

### İletişim formu üye yeni mesaj gönderme
```json
/srv/service/guest/send-contact-member-message
// departmentId,mobile_phone,email,surname,name,message

```
### İletişim formu yeni mesaj gönderme
```json
/srv/service/guest/send-contact-message
// DepartmanId,Ad,Soyad,EPosta,Cep,Mesaj,

```
### Ebülten aboneliği

```json
/srv/service/guest/subscribe-newsletter/
// post   name,surname, phone, email, kvkk, gender
```
### Bildirim  alma 

```json
/srv/service/guest/register-notification-list/
```
### Aboneliği aktif etme

```json
/srv/service/guest/activate-subscription/{TOKEN}

```
### Yorum yapma

```json
/srv/service/guest/comment/
// post text, rate, 

```


### Gönderilecek form oluşturma

```json
/srv/service/guest/form

```

### Oylama 

```json
/srv/service/guest/vote/{BLOCK_ID}{POLL_ID}{OPTIONS}

```
### Telefonla Sipariş 

```json
/srv/service/guest/order-by-telephone/

post  fullname, information, product_name, hour, date, business_phone, phone email

```
### ip öğrenme 
```json

/srv/service/guest/ip
```
