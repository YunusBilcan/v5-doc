## Ürün liste stoktatakiler
Ürün liste stokta var stokta yok için bu yapı kullanılabilir.


```html title="smarty "

<div class="col-auto d-none d-md-block">
    <div class="position-relative">
        <input type="checkbox" id="product-list-stock" class="form-control" {if $smarty.get.stock == 1} checked value="" {else} value="1" {/if} onchange="window.location.href = T.getLink('stock', this.value)">
        <label for="product-list-stock" class="m-0 d-flex align-items-center fw-regular">
            <span class="input-checkbox"><i class="ti-check"></i></span>{#in_stocks#}jkk
        </label>
    </div>
</div>

```
